# HILT

Hilt fournit une méthode standard pour intégrer l'injection de dépendances Dagger dans une application Android.

Les objectifs de Hilt sont les suivants :

Simplifier l'infrastructure liée à Dagger pour les applications Android.
Créer un ensemble standard de composants et de champs d'application afin de faciliter la configuration, la lisibilité/compréhension et le partage de code entre les applications.
Fournir un moyen simple de provisionner différentes liaisons pour divers types de compilation (par exemple, test, débogage ou publication).
